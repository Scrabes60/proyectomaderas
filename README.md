# Maderas Carrera

Landing page profesional para distribuidor de madera y tableros.

## Deploy en Vercel

### Paso 1: Sube el codigo a GitHub
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/TU-USUARIO/maderas-carrera.git
git push -u origin main
```

### Paso 2: Conecta con Vercel
1. Ve a [vercel.com](https://vercel.com)
2. Haz clic en **"Add New Project"**
3. Selecciona tu repo `maderas-carrera`
4. Vercel detecta automaticamente que es Next.js
5. Haz clic en **"Deploy"**
6. Listo! Tu sitio estara en: `https://carreramaderas.vercel.app`

## Desarrollo local

```bash
npm install
npm run dev
```
